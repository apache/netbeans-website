<div class="sect2">
<h3 id="_how_do_i_use_matisse_grouplayout_new_form_editor_layout_manager_in_5_0_in_my_window_system_components">How do I use Matisse/GroupLayout (new form editor/layout manager in 5.0) in my window system components?</h3>
<div class="paragraph">
<p>When you create a new window system component (<a href="DevFaqWindowsTopComponent.asciidoc">TopComponent</a>) using the template wizard in NB 5.0 or greater, the default layout manager is <code>GroupLayout</code> (the new, super-easy-to-design-with layout manager in NB 5.0). So you do not need to do anything special. Your module will have a declared dependency on the corresponding library module which is in the NB Platform as of 5.0.</p>
</div>
<div class="paragraph">
<p><code>GroupLayout</code> is included in JDK 6; as long as JDK 5 needs to be supported (until JDK 8 is released at a minimum), NetBeans will use the library version rather than the JDK 6 version.</p>
</div>
<hr>
<div class="paragraph">
<p>Applies to: NetBeans 5.0, 5.5, 6.x</p>
</div>
</div>
