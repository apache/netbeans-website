<h2 id="_where_can_i_find_javadoc_for_the_ide_and_platform">Where can I find Javadoc for the IDE and Platform?</h2>
<div class="sectionbody">
<div class="paragraph">
<p>There is no separate set of Javadoc for the NetBeans Platform.  However, as the Platform is just a subset of the IDE, the Javadoc for the IDE will apply.</p>
</div>
<div class="paragraph">
<p>You can <a href="http://bits.netbeans.org/dev/javadoc/index.html">browse the Javadoc online</a> (this link always points to the latest development version).</p>
</div>
<div class="paragraph">
<p>You can also download the Javadoc for a particular NetBeans release:</p>
</div>
<div class="olist arabic">
<ol class="arabic">
<li>
<p>Go to the nightly build download site: <a href="http://bits.netbeans.org/dev/nightly/">http://bits.netbeans.org/dev/nightly/</a></p>
</li>
</ol>
</div>
<div class="olist arabic">
<ol class="arabic" start="2">
<li>
<p>Click the link for the build you want.</p>
</li>
</ol>
</div>
<div class="olist arabic">
<ol class="arabic" start="3">
<li>
<p>Add "zip/" to the end of the URL in your browser&#8217;s address bar and hit enter. In other words, the complete URL might look like this: <a href="http://bits.netbeans.org/dev/nightly/2010-06-11_00-01-15/zip/">http://bits.netbeans.org/dev/nightly/2010-06-11_00-01-15/zip/</a></p>
</li>
</ol>
</div>
<div class="olist arabic">
<ol class="arabic" start="5">
<li>
<p>There are about a dozen links on that page. The one you want begins with <code>netbeans-trunk-nightly</code> and ends with <code>-javadoc.zip</code>; click that link to download the Javadoc archive.</p>
</li>
</ol>
</div>
<div class="paragraph">
<p>Finally, you can go to the update center (<strong>Tools &gt; Plugin Manager</strong>) in the NetBeans IDE and request the <strong>NetBeans API Documentation</strong> module, which bundles Javadoc matching your IDE release. Use <strong>Help &gt; Javadoc References</strong> (<strong>View &gt; Documentation Indices</strong> in NetBeans 5.x) to see the overview pages for API sets currently used by your open projects. The IDE should also automatically display this Javadoc in code completion popups.</p>
</div>
<div class="paragraph">
<p>Applies to: NetBeans 5.x, 6.x, 7.x</p>
</div>
<div class="paragraph">
<p>Platforms: all</p>
</div>
</div>
</div>
