<div class="sect2">
<h3 id="_what_is_a_layer_file_or_module_layer">What is a layer file, or "module layer"?</h3>
<div class="paragraph">
<p>Layer files are small XML files provided by modules, which define a virtual filesystem (<a href="DevFaqFileSystem.asciidoc">DevFaqFileSystem</a>).
The layer file defines folders and files that will be merged into the system filesystem (<a href="DevFaqSystemFilesystem.asciidoc">DevFaqSystemFilesystem</a>)
that makes up the runtime configuration information NetBeans and its modules use.</p>
</div>
<div class="paragraph">
<p>Layer files help to make it possible for modules to be dynamically installed.
If you&#8217;ve read about `FileObject`s (<a href="DevFaqFileObject.asciidoc">DevFaqFileObject</a>)
and `FileSystem`s (<a href="DevFaqFileSystem.asciidoc">DevFaqFileSystem</a>),
you know that you can listen for changes in folders and files in a filesystem.
That&#8217;s exactly what the components of NetBeans whose content is composed from folders in the system filesystem do.
So if a module is added at runtime, the system filesystem fires changes;
the UI notices that the contents of the folder has
changed and updates the UI to reflect the changes.</p>
</div>
<div class="paragraph">
<p>If you created your module using the IDE, you may already have an XML layer in your
module, and you can expand the node for it under <strong>Important Files</strong> in your module project to see and modify its contents.
The way it is declared is simple:</p>
</div>
<div class="ulist">
<ul>
<li>
<p>In your JAR, provide the layer file - e.g. <code>com/foo/mymodule/resources/layer.xml</code></p>
</li>
<li>
<p>In your module&#8217;s manifest, include the following line somewhere in the top section:</p>
</li>
</ul>
</div>
<div class="listingblock">
<div class="content">
<pre>OpenIDE-Module-Layer: com/foo/mymodule/resources/layer.xml</pre>
</div>
</div>
<div class="paragraph">
<p>Some Java source code annotations generate layer entries for you (you do not need to have a <code>layer.xml</code> in your module&#8217;s source tree).</p>
</div>
<div class="paragraph">
<p>&lt;hr/&gt;
Applies to: NetBeans 6.7 and later</p>
</div>
</div>
