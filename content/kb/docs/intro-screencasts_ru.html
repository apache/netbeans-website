Обзор IDE NetBeans 8.0

+ +

*NetBeans IDE 8.0* обеспечивает полную поддержку новейших технологий Java 8 — Java SE 8, Java SE Embedded 8 и Java ME Embedded 8. Кроме того, IDE предоставляет ряд улучшений, расширяющих поддержку Maven и Java EE с PrimeFaces, новые инструменты для HTML5, в частности для AngularJS, и обеспечивает усовершенствованную поддержку PHP и C/C+.

Посмотрите link:ide/overview-screencast.html[Видеообзор NetBeans IDE 8.0+] чтобы узнать о новых возможностях последней версии.
