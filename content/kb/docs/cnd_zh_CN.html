开发 C、C++ 和 Fortran 应用程序

当安装支持 C/C 的 NetBeans IDE 时，IDE 中将包括 C 和 C 项目类型以及相应的模板。您可以使用动态库和静态库创建 C、C 和 Fortran 应用程序，也可以创建基于现有源代码的 C/C/Fortran 项目。编辑器可以缩进、补充完整 C 和 C++ 代码并能突出显示其语法，它很好地集成了 GNU gdb 调试器。
