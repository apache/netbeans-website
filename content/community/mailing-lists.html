Mailing lists are the main way to get in touch with the NetBeans Community. Feel free to subscribe and participate.

Lists are managed with ezmlm, refer to ezmlm manual for useful help commands.

The mailing lists can also be subscribed to in digest form, so that you at most receive one e-mail per day. Digest form email addresses are the same as the email addresses below, substituting subscribe@ with digest-subscribe@ and substituting unsubscribe@ with digest-unsubscribe@.

Users list

Use this list to ask questions about the usage of the NetBeans IDE, whether for Java, JavaScript, C/C++, or any other language or technology, whether as user of NetBeans IDE or developer on top of the NetBeans Platform.

This list is also useful as well for those using the NetBeans APIs to develop modules and applications.

Using prefix tags in the subject line, such as [ java ] or [ javascript ] is encouraged, to clearly distinguish the topic you’re addressing in your e-mail.

  • Mailing list address: users@netbeans.incubator.apache.org

  • To subscribe please send an email to: users-subscribe@netbeans.incubator.apache.org

  • To unsubscribe please send an email to: users-unsubscribe@netbeans.incubator.apache.org

  • See the Apache mailing list archive

  • Pony mail archive

Dev list

Specifically for those committed to working on the development of Apache NetBeans itself, as well as those committed to promoting and documenting it, via events and tutorials, etc.

  • Mailing list address: dev@netbeans.incubator.apache.org

  • To subscribe please send an email to: dev-subscribe@netbeans.incubator.apache.org

  • To unsubscribe please send an email to: dev-unsubscribe@netbeans.incubator.apache.org

  • See the Apache mailing list archive

  • Pony mail archive

NetCAT list

Specifically for those involved in the NetBeans Community Acceptance Testing (NetCAT) program.

  • Mailing list address: netcat@netbeans.incubator.apache.org

  • To subscribe please send an email to: netcat-subscribe@netbeans.incubator.apache.org

  • To unsubscribe please send an email to: netcat-unsubscribe@netbeans.incubator.apache.org

  • See the Apache mailing list archive

  • Pony mail archive

Announce list

Receive announcements only, e.g., when a new release is made available, i.e., for those who do not want to get much traffic though want to keep up with the high level developments of Apache NetBeans. No one other than administrators are able to write to this mailing list.

  • Mailing list address: announce@netbeans.incubator.apache.org

  • To subscribe please send an email to: announce-subscribe@netbeans.incubator.apache.org

  • To unsubscribe please send an email to: announce-unsubscribe@netbeans.incubator.apache.org

  • See the Apache mailing list archive

Commits list

  • Mailing list address: commits@netbeans.incubator.apache.org

  • To subscribe please send an email to: commits-subscribe@netbeans.incubator.apache.org

  • To unsubscribe please send an email to: commits-unsubscribe@netbeans.incubator.apache.org

  • See the Apache mailing list archive

Notifications list

  • Mailing list address: notifications@netbeans.incubator.apache.org

  • To subscribe please send an email to: notifications-subscribe@netbeans.incubator.apache.org

  • To unsubscribe please send an email to: notifications-unsubscribe@netbeans.incubator.apache.org
