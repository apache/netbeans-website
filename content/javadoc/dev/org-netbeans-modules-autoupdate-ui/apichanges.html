    Autoupdate UI API

    PluginManager.install

    Feb 13 '13; API spec. version: 1.35; affected top-level classes: PluginManager; made by: phejl; issues: #196538

    PluginManager ands its installSingle and install shows standard UI for easy installation of single plugin based on codenamebase.

    PluginManager.openInstallWizard

    Feb 15 '11; API spec. version: 1.21; affected top-level classes: PluginManager; made by: jtulach; issues: #194518

    PluginManager ands its openInstallWizard shows standard UI for installing of selected modules.
